前端页面组成由那几部分

1.结构层 html
2.表示层 css
3.行为层 javascript(js)属于弱类行语言

ECMAScript（理解为javascript的一个标准）

样式分为三种

1.行内样式（内联样式） 写在结构上
2.内部样式 写在html文件内
3.外部样式

position定位

position 分三种 绝对，相对，固定
绝对定位 参照物是 最近的祖先定位元素 不保留自己的位置
相对定位 是相对自己定位 保留自己的位置
固定定位 理解为小广告

margin和padding的四位写法

1px 2px 上下 左右
1px 2px 3px 上 左右 下
1px 2px 3px 4px 上 右 下 左

清浮动 三种方法

1.空标签 在css中写clear  在最后一个浮动元素写  缺点：代码太多 有点：方便，快捷
2.overflow:hidden(在css样式中给父级加)  缺点：如果div有点位的时候,会显示不全
3.clearfix (在css中，在结构层中结构父级起名) 最快捷省事的方法

背景图相关

背景图 是不经常更换的 
box-shadow: 水平 垂直 模糊距离 阴影尺寸 颜色 内外部的切换（默认是外部的（选框阴影）
text-shadow: 水平 垂直 模糊距离  颜色（文字阴影）属性的可以继承的
border-radius:单位 （圆角） 
background:linear-gradient (颜色，颜色) 渐变
background-size:单位 单位  （背景图尺寸）
background-size:cover（把背景图像扩展至足够大，以使背景图像完全覆盖背景区域。背景图像的某些部分也许无法显示在背景定位区域中。）
background-size：contain;（把图像图像扩展至最大尺寸，以使其宽度和高度完全适应内容区域。）

background-origin: padding-box|border-box|content-box;（背景图像的位置关系）对应img
background-clip: border-box|padding-box|content-box;（属性规定背景的绘制区域）对应bgi

2d转换，3d转换 动画

translate() 从当前位置移动

角度的单位是deg

transform:translate(单位，单位) 位置移动

transform:rotate(度数，写作deg) 旋转

@keyframes rotate
0%{
	transform:rotate(0deg)
}
50%{
	transform:rotate(180deg)
}
100%{
	transform:rotate(360deg)
}

transform:scale(0.5,0.5) 缩放 （保留原来的位置，不影响其他的元素）

animation: rotate 5s linear infinite;动画（语法：animation: name duration timing-function delay iteration-count direction;）

animation-name	规定需要绑定到选择器的 keyframe 名称。。
animation-duration	规定完成动画所花费的时间，以秒或毫秒计。
animation-timing-function	规定动画的速度曲线。
animation-delay	规定在动画开始之前的延迟。
animation-iteration-count	规定动画应该播放的次数。
animation-direction	规定是否应该轮流反向播放动画。

transform-orlgin 改变被转换元素的位置
skew(角度，角度) 元素翻转给定的角度

transform-matrix   把所有 2D 转换方法组合在一起。需要六个参数（x轴伸缩，y轴伸缩，x轴翻转，y轴翻转，x轴移动，y轴移动）



transition 过度  四个值（transition-property，transition-duration，transition-timing-function，transition-delay）

transition-property	规定应用过渡的 CSS 属性的名称。
transition-duration	定义过渡效果花费的时间。
transition-timing-function	规定过渡效果的时间曲线。
transition-delay	规定过渡效果何时开始。

transition:all 时间； （all是全部的意思）（这是 一种组合的写法）



transform (2D转换 后面接5个值 translate（位移） rotate（旋转） scale（缩放） skew(翻转) matrix（全部方法的组合）)

windows serve的用法

serve只需要使用node -v命令即可查看nodejs的版本信息。

1.安装 node（禁止修改路径） 
2.安装 git bash 命令行 
3.打开 git bash 使用  npm i -g serve 安装服务包
4.到项目文件夹内 打开 git bash  输入命令 serve .
5.在浏览器（pc或者手机）中输入电脑的ip地址加：5000 就可以查看本地文件夹中写的东西了
